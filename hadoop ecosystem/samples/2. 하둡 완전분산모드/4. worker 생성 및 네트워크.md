### 단계6:  모두 실행 > Hosts 설정
- 각 프라이빗 IPv4 주소 복사 및 적용 
```shell
# hosts 파일 편집
sudo vim /etc/hosts

# 아래 내용처럼 ip주소 추가 후 저장
10.0.2.15 master # 프라이빗 IPv4 주소 복사
10.0.2.11 worker1 # 프라이빗 IPv4 주소 복사
10.0.2.10 worker2 # 프라이빗 IPv4 주소 복사
10.0.2.12 worker3 # 프라이빗 IPv4 주소 복사
```
