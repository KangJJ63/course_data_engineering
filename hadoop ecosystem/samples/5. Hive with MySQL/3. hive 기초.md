# [Table](https://www.javatpoint.com/hive-create-table)

---
### 단계1: database 생성 및 적용 
```shell
create database if not exists demo; # 생성
use demo; # 적용 
```
![alt text](image-8.png)

---
### 단계2: table 생성 
```shell
create table employee (Id int, Name string , Salary float)
```
