---
- Hosts 설정
  - 각 프라이빗 IPv4 주소 복사 및 적용 
```shell
# hosts 파일 편집
sudo vim /etc/hosts

# 아래 내용으로 추가 후 저장
10.0.2.15 master # 프라이빗 IPv4 주소 복사
10.0.2.4 worker1 # 프라이빗 IPv4 주소 복사
10.0.2.5 worker2 # 프라이빗 IPv4 주소 복사
10.0.2.6 worker3 # 프라이빗 IPv4 주소 복사
```
