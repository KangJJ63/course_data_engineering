### 단계5: 모두 실행 > sshd_config 수정 
```shell
vim /etc/ssh/sshd_config

# sshd_config파일에서 아래와 같이 수정 
PermitRootLogin yes # 주석 제거 및 yes로 수정 
PasswordAuthentication yes # no를 yes로 수정 
```
